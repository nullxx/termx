## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nullxx/termx.git
   ```
2. Install NPM packages
   ```sh
   yarn
   ```
3. Run it
   ```sh
   yarn run electron:dev
   ```
